🔗 Connexion GitHub Gist

GitHub Gist sert de base de données — gratuit, 100% dans GitHub. Tout le monde voit les résultats en temps réel.

1Va sur gist.github.com → Crée un nouveau Gist public → Nom du fichier : tournament.json → Contenu : {}
2Copie l'ID du Gist depuis l'URL (ex: abc123def456)
3Crée un Token GitHub : Clique ici → scope gist → Generate → Copie-le (pour toi seul, pas les spectateurs)
Les spectateurs laissent le token vide — ils verront les résultats mais ne pourront pas modifier.
